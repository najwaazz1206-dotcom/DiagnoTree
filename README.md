# DiagnoTree - Expert System

Aplikasi sistem pakar untuk deteksi dini penyakit berbasis **Depth-First Search (DFS) Tree**.

## Referensi Data Medis
Logika diagnosa dalam aplikasi ini disusun berdasarkan referensi berikut:
1. **Gejala DBD:** Panduan Praktik Klinis Kemenkes RI No. HK.02.02/2015.
2. **Gejala COVID-19:** Protokol Kesehatan WHO (World Health Organization) 2022.
3. **Gejala Tifoid:** Buku Ajar Ilmu Penyakit Dalam UI (Edisi VI).

*Catatan: Aplikasi ini hanya untuk deteksi dini dan bukan merupakan diagnosa medis final.*
